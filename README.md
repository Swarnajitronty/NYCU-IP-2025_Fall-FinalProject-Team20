# NYCU-IP-2025_Fall-FinalProject-Team20
“Lunar Lumina” is a Fall 2025 image-processing project showcasing a non-AI pipeline for enhancing lunar astrophotography using stacking, wavelet denoising, CLAHE, and sharpening. The method improves SNR, PSNR, and SSIM efficiently using classical techniques with full code, results, and visuals included.
